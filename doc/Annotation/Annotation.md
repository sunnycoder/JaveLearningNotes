## 概述

什么是注解？
    
    Java 注解用于为 Java 代码提供元数据。
    作为元数据，注解不直接影响你的代码执行，但也有一些类型的注解实际上可以用于这一目的。
    Java 注解是从 Java5 开始添加到 Java 的。
